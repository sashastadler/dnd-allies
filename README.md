# dnd-allies
A program for handling/tracking the homebrew Allies in our dnd campaign

## Running dnd-allies
More info to come once release is published.

## Adding custom allies
1. Make a copy of one of the existing allies in the "characters" folder.
2. Rename the file to the name of the ally.
    This will be the name that shows up in the list of allies. Underscores are automatically replaced with spaces.
3. Open the .json file and change the values to be whatever needed for your character. It's important to maintain the overall json formatting, so do not add new fields or rename them.
4. (Optional) Add an image with the same name as the .json you made to the `characters/images` folder. This doesn't do anything yet but one day I'll add loading images too.
